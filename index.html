Built for lecture speedrunning.

The updated extension UI is less “landing page demo” and more polished product: compact, clearer, and focused on the actions users actually need.

⚡

Custom playback speed

Use the slider for normal changes, or click the green speed badge and type a custom value directly.

🔁

Startup speed

Choose whether the saved speed should apply automatically when a Panopto page loads.

⏱️

Time saved tracking

The extension tracks saved time while watching faster than normal, and avoids counting manual seeking as progress.

🏆

Leaderboard

Users can sign in and opt into leaderboard logging to compare time saved with other students.

🌍

Multiple languages

The popup supports English, Korean, Chinese, Hebrew, and Ukrainian UI labels.

🧩

Lightweight Chrome install

Add it to Chrome and use it directly on Panopto without changing your normal workflow.

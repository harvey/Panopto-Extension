Technical snapshot.

Useful if you want this page to show more than marketing copy.

Manifest V3 Chrome extensionRuns as a Chrome extension with a popup UI, background service worker, and content script.
Panopto domain targetingThe content script targets *.panopto.com and *.panopto.eu.
Playback speed injectionThe content script finds the page video element and sets video.playbackRate to the stored custom speed.
Startup speedWhen enabled, the extension applies the saved speed shortly after the Panopto page loads.
Time saved trackingThe extension tracks saved time while playback rate is above normal speed and avoids counting manual seeking as saved time.
Leaderboard conceptThe extension includes a login system/leaderboard flow so users can compare saved time.
